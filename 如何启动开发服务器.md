# 如何启动开发服务器

## 🤔 什么是开发服务器？

**开发服务器**是一个运行在您电脑上的程序，它：
- 让您的网站可以在浏览器中访问（通常通过 `http://localhost:3000`）
- 提供实时更新功能（修改代码后，浏览器自动刷新）
- 提供热重载功能（保存文件后立即看到效果）

**简单理解**：就像打开一个"本地网站"，而不是直接打开 HTML 文件。

## 🚀 如何启动开发服务器

### 步骤 1：打开终端

1. **在 Cursor 中打开终端**：
   - 按 `` Ctrl + ` ``（反引号键，通常在数字 1 左边）
   - 或者点击菜单：`终端` → `新建终端`

2. **或者在系统的终端应用中**：
   - Mac: 打开"终端"应用（Terminal）
   - 确保切换到项目目录：
     ```bash
     cd /Users/henryhuang/Documents/GitHub/personal-website-for-HENRY
     ```

### 步骤 2：运行启动命令

在终端中输入以下命令：

```bash
npm run dev
```

然后按 `Enter`（回车键）

### 步骤 3：等待启动

您会看到类似这样的输出：

```
▲ Next.js 14.2.25
- Local:        http://localhost:3000
- ready started server on 0.0.0.0:3000, url: http://localhost:3000
```

**看到这些信息就说明启动成功了！** ✅

### 步骤 4：在浏览器中打开

1. **打开浏览器**（Chrome、Safari、Firefox 等）
2. **访问以下地址**：
   ```
   http://localhost:3000
   ```
3. 您应该能看到您的网站了！

## 🎯 为什么要启动开发服务器？

**Stagewise 需要开发服务器才能工作！**

- Stagewise 扩展会连接到开发服务器
- 通过开发服务器，Stagewise 才能：
  - 检测到您在浏览器中点击的元素
  - 自动定位到对应的代码
  - 修改代码并实时更新

## ⚠️ 常见问题

### 问题 1：命令找不到（npm: command not found）

**原因**：没有安装 Node.js

**解决方案**：
1. 访问 [nodejs.org](https://nodejs.org/)
2. 下载并安装 Node.js（推荐 LTS 版本）
3. 安装完成后，重新打开终端
4. 再次运行 `npm run dev`

### 问题 2：端口已被占用（Port 3000 is already in use）

**原因**：另一个程序正在使用 3000 端口

**解决方案**：
1. 停止正在运行的开发服务器（在终端中按 `Ctrl + C`）
2. 或者使用其他端口：
   ```bash
   npm run dev -- -p 3001
   ```
   然后访问 `http://localhost:3001`

### 问题 3：依赖未安装

**错误信息**：`Cannot find module 'xxx'` 或类似

**解决方案**：
1. 在项目目录中运行：
   ```bash
   npm install
   ```
2. 等待安装完成后，再运行：
   ```bash
   npm run dev
   ```

## 🛑 如何停止开发服务器

在终端中按：
- **Mac/Linux**: `Ctrl + C`
- **Windows**: `Ctrl + C`

## 📝 其他有用的命令

- **启动开发服务器**：`npm run dev`
- **构建生产版本**：`npm run build`
- **启动生产服务器**：`npm start`
- **检查代码错误**：`npm run lint`

## ✨ 总结

1. 打开终端
2. 运行 `npm run dev`
3. 等待看到 "ready" 消息
4. 在浏览器中打开 `http://localhost:3000`
5. 现在可以使用 Stagewise 了！

**提示**：开发服务器启动后，终端会一直运行（保持打开状态），这样才能保持网站运行。当您需要停止时，按 `Ctrl + C` 即可。
