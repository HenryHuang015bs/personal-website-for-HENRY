# personal-website-for-HENRY
个人网站

## 开发工具

本项目支持 [Stagewise](https://marketplace.cursorapi.com/items/?itemName=stagewise.stagewise-vscode-extension) - 一个将 AI 编程助手与浏览器无缝连接的前端开发工具。

### 快速开始

1. **安装 Stagewise VS Code 扩展**
   - Cursor 会自动提示您安装推荐的扩展
   - 或访问 [扩展市场](https://marketplace.cursorapi.com/items/?itemName=stagewise.stagewise-vscode-extension) 手动安装

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **在浏览器中使用**
   - 访问 `http://localhost:3000`
   - 点击页面元素，通过 Stagewise 连接 AI 助手进行代码修改

详细使用说明请查看 [STAGEWISE_GUIDE.md](./STAGEWISE_GUIDE.md)
